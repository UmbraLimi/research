# Review

**Purpose:** Review and suggest improvements to a file.

## Actions

1. Read the specified file
2. Analyze for:
   - Completeness
   - Clarity
   - Consistency with project conventions
   - Missing information
3. Suggest specific improvements
4. Offer to make updates

## Example

```
/r-review /user-stories/story-001-user-login.md
```
