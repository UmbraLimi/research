# Capture Learning

**Purpose:** Document a specific learning or insight discovered during research.

## Actions

1. Check STRUCTURE.md for next learning number
2. Create `learning-NNN-[topic].md` in `/learnings/`
3. Document:
   - What we learned
   - Why it matters for Alpha Peer
   - Key insights
   - Implications for architecture/tech choices
   - References/sources
4. Update `learnings-index.md` with summary
5. Update STRUCTURE.md and session file

## Example

```
/r-capture-learning websocket scaling challenges
```
