# Research Technology

**Purpose:** Research a specific technology for potential use in Alpha Peer.

## Actions

1. Check STRUCTURE.md for next tech file number
2. Create `tech-NNN-[technology-name].md` in `/research/`
3. Research the technology (web search if needed)
4. Document:
   - Core capabilities and use cases
   - Integration complexity
   - Pricing model
   - Developer experience
   - Community and support
   - Relevance to Alpha Peer
5. Update STRUCTURE.md with new file number
6. Update session file with research completed

## Example

```
/r-research-tech supabase
```
