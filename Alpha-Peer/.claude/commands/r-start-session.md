# Start Session

**Purpose:** Begin a new work session with proper context loading.

## Actions

1. Read STRUCTURE.md for current state and conventions
2. Check what session number this is for today
3. Create session file in `docs/sessions/YYYY-MM/` with filename `YYYY-MM-DD_HH-MM-SS [Type].md`:
   - Session metadata (date, number, goals)
   - Link to previous session
   - Starting context
4. Ask: "What are your goals for this session?"
