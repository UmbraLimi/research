## Deferred Enhancements

### Feature Tracking Rule
**Trigger:** When project has enough docs that features could be lost
**Action:** Mention them in PLAN.md with cross-references

### Durable Insight Capture
**Trigger:** When DECISIONS.md has enough entries
**Action:** Capture qualifying insights as blockquotes alongside decisions

### Test Suite Workflow
**Trigger:** When first tests are written
**Action:** Run once, extract failures, present before fixing individually

### Generated Docs Must Have Single Regeneration Command
**Trigger:** When first derived doc is created
**Action:** Any derived doc must be regenerable with one command
